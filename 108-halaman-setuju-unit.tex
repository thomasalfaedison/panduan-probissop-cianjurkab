\section*{8. Menu Setuju}
\addcontentsline{toc}{section}{8. Menu Setuju}

\noindent Untuk mengakses Halaman Menu Setuju, pada daftar menu sebelah kiri klik Menu pada menu SOP seperti gambar berikut:
\begin{figure}[H]
      \centering
      \includegraphics[width=12.45cm]{media/Setuju-unit/1-menu-Setuju.png}
      \caption{Akses Menu Setuju}
      \label{fig:bagan-alir}
    \end{figure}
    \vspace{-2em}

Setelah diklik maka akan diarahkan ke halaman Menu Setuju seperti berikut:
\begin{figure}[H]
      \centering
      \includegraphics[width=12.45cm]{media/Setuju-unit/2-halaman-Setuju.png}
      \caption{Halaman Menu Setuju}
      \label{fig:bagan-alir}
    \end{figure}
    \vspace{-2em}

Pada Menu Setuju, ada beberapa fitur yang user dapat gunakan seperti Export Data Setuju, Filter Data Setuju, dan lihat detail data Setuju. Berikut tahapan setiap fitur pada Menu Setuju:

\subsection*{8.1 Export Setuju}
\addcontentsline{toc}{subsection}{8.1 Export Setuju}

Untuk Export Data Setuju, klik tombol Export Setuju, maka otomatis download Data Setuju.
\begin{figure}[H]
    \centering
    \includegraphics[width=12.45cm]{media/Setuju-unit/1-export-Setuju.png}
    \caption{Export Setuju}
    \label{fig:bagan-alir}
\end{figure}
