\section*{9. Menu Aktif}
\addcontentsline{toc}{section}{9. Menu Aktif}

\noindent Untuk mengakses Halaman Menu Aktif, pada daftar menu sebelah kiri klik Menu pada menu SOP seperti gambar berikut:
\begin{figure}[H]
      \centering
      \includegraphics[width=12.45cm, height=5.35cm]{media/Aktif-unit/1-menu-Aktif.png}
      \caption{Akses Menu Aktif}
      \label{fig:bagan-alir}
    \end{figure}
    \vspace{-2em}

Setelah diklik maka akan diarahkan ke halaman Menu Aktif seperti berikut:
\begin{figure}[H]
      \centering
      \includegraphics[width=12.45cm, height=5.35cm]{media/Aktif-unit/2-halaman-Aktif.png}
      \caption{Halaman Menu Aktif}
      \label{fig:bagan-alir}
    \end{figure}
    \vspace{-2em}

Pada Menu Aktif, ada beberapa fitur yang user dapat gunakan seperti Export Data Aktif, Filter Data Aktif, dan lihat detail data Aktif. Berikut tahapan setiap fitur pada Menu Aktif:

\addcontentsline{toc}{subsection}{9.1 Export Aktif}

Untuk Export Data Aktif, klik tombol Export Aktif, maka otomatis download Data Aktif.

    \caption{Export Aktif}
