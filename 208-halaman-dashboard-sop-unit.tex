\section*{2. Halaman Dashboard SOP}
\addcontentsline{toc}{section}{2. Halaman Dashboard SOP}

\noindent Untuk mengakses Halaman Dashboard SOP, pada daftar menu sebelah kiri klik Dashboard pada menu SOP seperti gambar berikut:
\begin{figure}[H]
      \centering
      \includegraphics[width=12.45cm, height=5.35cm]{media/dashboard-sop-unit/1-akses-dashboard-sop.png}
      \caption{Akses Dashboard SOP}
      \label{fig:bagan-alir}
    \end{figure}
    \vspace{-2em}

Setelah diklik maka akan diarahkan ke halaman Dashboard SOP seperti berikut:
\begin{figure}[H]
      \centering
      \includegraphics[width=12.45cm, height=5.35cm]{media/dashboard-sop-unit/2-dashboard-sop.png}
      \caption{Halaman Dashboard SOP}
      \label{fig:bagan-alir}
    \end{figure}
    \vspace{-2em}

Pada Menu Dashboard SOP, ditampilkan informasi jumlah Total SOP Unit, jumlah Konsep, jumlah Verifikasi, jumlah SOP Aktif, dan jumlah SOP pada Perangkat Daerah yang login. Berikut tahapan setiap fitur pada Menu Dashboard SOP:
